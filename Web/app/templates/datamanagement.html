Upload .csv files for CITA analysis


CSV files must meet the following criteria to be correctly processed:

  • The first line must contain column header names
  • Each column value must be contained within quotes
  • Quotes should be removed from the original data
  • Each column value must be separated by a comma
  • Commas in the original data are acceptable
  • The file should contain no empty lines or spaces outside the volumn values

Example:

"ID","ColumnName1","ColumnName2","ColumnName3"

"1","RowValue1","RowValue2","RowValue3"


Please email admin@CITA.com if clarification on .csv rules is required.
